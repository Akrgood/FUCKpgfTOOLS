# FUCKpgfTOOLS
![FUCKpgfTOOLS](https://img.shields.io/badge/FUCKpgfTOOLS-v4.0-red?style=for-the-badge&logo=python)

Editing and converting PGF fonts

!!!You need to install the following Python libraries:
1. pip install Pillow
2. pip install customtkinter
3. pip install freetype-py

This script allows you to redraw the PGF font used on the PSP™ (PlayStation Portable™).
It also includes a TTF/OTF to PGF converter created by my friend Ivan: https://github.com/r3trob0y/TTF-OTF-2-PGF

!!!Starting with version 3, I'm releasing ready-to-run EXE files that don't require any library installations or anything like that.
